# 🌐 Web Automation Basics with Selenium

This repository contains my hands-on practice and learning journey in web automation using Selenium WebDriver with Java. It covers the fundamental concepts, browser actions, locators, selectors, and WebElement interactions commonly used in automation testing.

The goal of this repository is to build a strong foundation in Selenium automation while documenting my progress through practical examples.

---

## 🛠 Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven

---

## 📚 Topics Covered

### 🔹 Browser Actions
- Open URLs
- Navigate between pages
- Refresh pages
- Browser window management
    - Maximize
    - Minimize
    - Fullscreen
- Multiple tabs and windows
- Window handles

### 🔹 Locators
- ID
- Name
- Class Name
- Tag Name
- Link Text
- Partial Link Text

### 🔹 Selectors
#### CSS Selectors
- Basic CSS selectors
- Parent → Child selection
- `first-child`
- `last-child`
- `nth-child`

#### XPath
- Basic XPath
- Attribute-based XPath
- Parent → Child XPath
- Indexed elements
- `last()` function

### 🔹 WebElement Interactions
- `click()`
- `sendKeys()`
- `clear()`
- `getText()`
- `getAttribute()`
- `getCssValue()`
- `isDisplayed()`
- `isSelected()`
- `isEnabled()`
